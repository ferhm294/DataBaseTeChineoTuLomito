# TeChineoTuLomito
## Descripción

Este proyecto consiste en la creación de una base de datos relacional para la gestión de una clínica veterinaria. La base de datos incluye diversas entidades relacionadas con las personas, clientes, empleados, mascotas, procedimientos y el registro de procedimientos aplicados. Además, se insertan datos iniciales en algunas tablas para facilitar pruebas y consultas.

## Tecnologías Utilizadas

- **Base de datos:** SQL Server
- **Entorno de desarrollo:** SQL Server Management Studio (SSMS)
